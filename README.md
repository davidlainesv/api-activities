# REST API Activities application

## Requirements
1. NodeJS==19.7.0

## Install dependencies
<code>npm install</code>
<code>npm install -g apidoc</code>

## Run the app
<code>npm run start</code>

## Run the app in development mode
<code>npm run dev</code>
